# MentorMinds — DIGITEX 2026 e-poster

Final submission files:
- **MentorMinds-DIGITEX2026-poster.pptx** — A1 portrait, built on the official
  DIGITEX 2026 template frame (ready to submit/print).
- **MentorMinds-DIGITEX2026-poster.png** — 2245×3179 raster (same artwork).
- **poster.html** + **render.mjs** — source; re-render with
  `node render.mjs` (needs Playwright Chromium).

Design follows award-winning e-poster conventions (3-30-300 rule, ~60%
visual / 40% text, clear hierarchy): a real MentorMinds app screen as the
hero, real product features as sticker callouts, and headline stats.
